# Contract Obligation Reminder System

## 1. Project Title

Contract Obligation Reminder System

---

## 2. Problem Statement

Organizations handle multiple contracts containing obligations, payment terms, renewal clauses, and notice periods. Manual tracking can lead to missed deadlines, compliance issues, and financial risks.

This project automates contract obligation tracking and reminder generation.

---

## 3. Dataset / Reference Source

Dataset Files:

- contracts.txt
- obligations.csv

The dataset contains sample contract obligations including:

- Payment Obligations
- Renewal Obligations
- Notice Period Obligations
- Due Dates
- Responsible Persons

## 6. AI / ML / Software Component

The AI module uses Python-based text processing and regular expressions to:

- Extract obligations
- Identify due dates
- Detect responsible persons
- Generate obligation summaries
- Assign risk levels
